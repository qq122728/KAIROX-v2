# FluxPerpDex 当前进度总结

更新时间：2026-05-26

## 项目定位

当前项目是一个本地模拟交易平台，不接真实撮合市场，主要用于自己测试和演示。

- 前台：移动端英文界面，包含登录、注册、行情、二元期权交易、资产、充值、提现、KYC、个人中心。
- 后台：中国后台管理界面，包含 Dashboard、存款地址、充值审核、提现审核、KYC 审核、用户管理、订单管理、市场管理、系统设置。
- 后端：Next.js API Routes + SQLite 本地数据库。
- 实时通信：已加入独立 Socket.IO 服务结构，用于后续后台和用户端实时刷新。
- 行情：前台已接入 Binance 公开行情方向，并保留本地 fallback。
